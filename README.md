# Сайт с золотыми цитатами Джейсона Стетхэма

Сайт для создания и просмотра цитат.  
Проект создан на **React**, использует **Firebase Realtime Database** и **React Router**.

---

## Функционал

- Добавление новой цитаты 
- Удаление цитаты
- Просмотр всех цитат  
- Просмотр цитат по категориям
- Toast-уведомления  
- Preloader при загрузке данных
- Пагинация

---

## Технологии

- React  
- React Router DOM  
- Axios  
- Firebase Realtime Database  
- CSS
